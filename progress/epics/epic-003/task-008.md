## 任务描述

在 Query 前端页面添加"保存到 Wiki"按钮

## 具体要求

1. **按钮显示条件**
   - 当 LLM 建议存入 wiki 时显示
   - 或者用户手动点击"保存"

2. **交互流程**
   - 点击按钮 → 显示确认对话框 → 确认后存入 wiki
   - 存入成功后显示提示

3. **UI 设计**
   - 按钮样式与现有风格一致
   - 带有保存图标或文字

## 验收标准

- [ ] Query 页面有"保存到 Wiki"按钮
- [ ] 点击后弹出确认对话框
- [ ] 确认后成功保存到 wiki/
- [ ] 成功后显示成功提示
