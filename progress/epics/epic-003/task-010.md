## 任务描述

基于当前查询或浏览的条目，推荐相关问题

## 具体要求

1. **基于当前查询推荐**
   - 用户输入问题后
   - LLM 推荐 3-5 个相关问题

2. **基于浏览的条目推荐**
   - 用户查看某个 wiki 条目时
   - 推荐可能感兴趣的相关话题

3. **推荐逻辑**
   - 基于 tags 相似度
   - 基于 linked 条目
   - 基于 LLM 语义理解

## 验收标准

- [ ] 查询后可获取相关问题推荐
- [ ] 浏览条目时可获取相关推荐
- [ ] 推荐问题点击后可执行查询
